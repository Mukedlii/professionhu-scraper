# 💼 Profession.hu Job Scraper

Automatikusan gyűjt állásajánlatokat a profession.hu-ról.

## Mit gyűjt?
- 📋 **Pozíció neve**
- 🏢 **Cég neve**
- 📍 **Helyszín**
- 💰 **Fizetés** (ha meg van adva)
- 🔗 **Link az álláshoz**
- 🕐 **Feladás dátuma**

## Felhasználási területek
- HR cégek számára tömeges állásgyűjtés
- Fizetési trendek elemzése
- Kereslet-kínálat figyelése különböző pozíciókra
- Versenytárs elemzés
